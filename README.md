云纹框架 V0.3
===============

> 运行环境要求主流浏览器或IE9及以上

## 基础信息

* 项目定位开发辅助框架

## 基础部署

~~~
// 加载基础框架
<script type="text/javascript" src="./moire/moire.min.js"></script>
// 加载扩展插件
<script type="text/javascript" src="./moire/moire.plugin.min.js"></script>
~~~

## 基础插件依赖

~~~
Jquery v1.9及以上
~~~

## 扩展插件依赖

~~~
MD5 0.7.3及以上
Layui v2.6.4及以上
Jquery v1.9及以上
Jquery_Resize
Jquery_Slick v1.6.0及以上
Jquery_Toast
Jquery_Viewer v1.5.0及以上
Imagesloaded v4.1.4及以上
masonry v4.2.2及以上
Sortable v1.10.2及以上
Viewer v1.5.0及以上
LayX v2.5.4及以上
~~~

## 打包依赖

~~~
npm
glob
uglify-js
~~~
